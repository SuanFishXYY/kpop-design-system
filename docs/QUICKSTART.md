# Quickstart - 5 minutes

## 1. What is this?

KPOP Design System turns K-pop visual strategy into executable council engines. It works in Claude/Copilot/Cursor out of the box - zero API key, zero config - because the host AI runs the protocol natively.

## 2. Install

```bash
git clone https://github.com/SuanFishXYY/kpop-design-system.git
```

## 3. Run first demo

```bash
npm test
node examples/council-ive-comeback.mjs
```

In Claude/Copilot/Cursor, load the skill and call `/kpop`; no provider key is required.
